胖国政府
Description
胖子国建立了一套新的政治体系，具体为有N个人，每个人可以传达命令给其他某些人，（不排除某些时候自己传达的命令又传回自己），每个人在心里都有一个默认的命令传达对象，即他接到命令或者想传达命令时只会传递给他默认的对象，如果他想改变自己的默认传达对象，他需要吃一顿烤肉。
胖A想传达命令给胖B，请你计算所有人总共最少需要吃几顿烤肉，A的命令可以传达给B。
Input
第一行有一个整数T，表示T组数据。
对于每组数据，第一行有3个整数2<=N<=100，1<=A，B<=N，分别表示人数，胖A和胖B的编号。
接下来有N行，每行的开头有一个数字Ki(0<=Ki<=N-1)，表示这个人可以传达命令给Ki个人，接下来有Ki个数字表示传递的人的编号，每个人默认传递对象为第一个数字所代表的人。
Output
对于每组数据输出一个数字，表示所有人总共最少需要吃几顿烤肉，A的命令可以传达给B，若无法从A传递到B，输出-1。
Sample Input 1 
13 2 12 2 32 3 12 1 2
Sample Output 1
0
Hint
包老板强无敌(破音)

题解：
题解：转换为最短路的问题。如果是默认联通的，就代表路上权值为0， 如果不是路的权值就为1
数据小，floyd可解
